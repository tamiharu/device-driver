# device-driver
2021年度ロボットシステム学の課題１で作成したデバイスドライバです。
講義の内容通り、1を入力するとLEDが点灯し、0を入力すると消灯します。

# 動作環境
*・raspberry pi 4B*
*・OS ubuntu 20.04 server*

# 用意した物
*・raspberry pi 4B × 1
・LED ×　1
・抵抗　240Ω
・ブレッドボード　
・ジャンパー線　(オス-メス)× 2*
